麒麟扫描软件是一款用\+Qt开发的界面友好的扫描软件。该软件可以根据扫描设备本身的分辨率、尺寸和色彩模式进行扫描， 同时增加了对扫描图片的后期处理（包括一键美化、智能纠偏和文字识别三大功能）。其他的一些图像处理小技巧也可以在此软件 上体现，如裁切、旋转等。
